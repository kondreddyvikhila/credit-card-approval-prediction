# 💳 Credit Card Approval Prediction System

This project is a Machine Learning-based system that predicts whether a credit card application will be approved or rejected based on applicant details. It automates the decision-making process using classification algorithms and helps reduce manual verification effort.

## 🧠 Problem Statement
Banks receive large numbers of credit card applications daily. Manual verification is time-consuming and inefficient. This project uses Machine Learning to predict approval decisions automatically.
